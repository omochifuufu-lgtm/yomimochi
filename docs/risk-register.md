## 見送った対策（理由つき）

- CSP(Content-Security-Policy)ヘッダ: GitHub Pagesは任意ヘッダを設定できない。metaタグCSPは可能だが、インラインscript/styleの全面改修が必要で費用対効果が低い（S1/S2対策で実害経路は塞いだ）
- 問い合わせフォーム: サーバーレス方針と衝突。当面はSNS（もっちー導線）で代替
