# 📚 よみもち（旧名: よみよみ きろく）

保育園・幼稚園〜小学生向けの、子ども用 読書記録アプリです。
タブレットやスマホのブラウザで動く Web アプリ（PWA 的な作り）。
**サーバー・ログイン不要、データはすべて端末の中に保存**されます。

## できること

| 機能 | 説明 |
|------|------|
| 📷 バーコードで記録 | 本の裏の ISBN バーコードをカメラで読み取り、書名・著者・表紙・出版社を自動取得（`openBD` / Google Books） |
| 🔢 手入力 | カメラが使えない時は ISBN 番号 / 書名で検索 |
| 📖 よみきかせカウント | 「よんだ！」ボタンで回数を +1。**はじめて読んだ日 / 最近読んだ日 / 累計回数**を記録 |
| ✏️ 感想を残す | **手書き（指でお絵かき）・音声（しゃべると文字化）・キーボード**の3通り |
| 🎯 目標と達成グラフ | 1週間 / 1か月 / 1年 の目標冊数を設定。あと何冊で達成かをバーで表示 |
| 📊 統計 | きょう / こんしゅう / ことし / 生涯 の読んだ冊数を集計 |
| 👦👧 きょうだい対応 | 子どもごとにプロフィール（名前＋アバター）を作成・切り替え。本棚・回数・感想・目標・統計をそれぞれ分離 |
| 💾 バックアップ | 全員分の記録を JSON ファイルに書き出し／復元（端末買い替え・機種変更時の引っ越しに） |
| 🏅 ごほうびスタンプ | 「はじめの1さつ」「50さつ」「同じ本10回」「感想10こ」「目標達成」など12種のマイルストーンでスタンプを獲得。達成した瞬間にお祝い演出が出る |

## つかいかた（起動）

カメラ機能は `file://` では動かないため、簡単なローカルサーバーで開きます。

```bash
cd "yomimochi"
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開く。
タブレット/スマホの実機（カメラあり）で試すのがおすすめです。

> カメラなしのPC・このプレビューでも、ISBN手入力や書名検索・記録・感想・目標などは全部動きます。

## 仕組み（技術メモ）

- **書籍DB**: [openBD](https://openbd.jp/)（日本の書籍・無料・APIキー不要）→ 見つからなければ Google Books API
- **バーコード**: ブラウザ標準の `BarcodeDetector` API（EAN-13 / 978・979 始まりを抽出）
- **音声**: Web Speech API（`ja-JP`）
- **手書き**: Canvas（PNGとして保存）
- **保存**: `localStorage`（キー `yomiyomi.v1`）

## 🌍 公開して収益化する（SNS拡散・アフィリエイト）

このアプリには「おうちの人へ」エリアに**収益導線**と**SNSシェア**が組み込まれています。
収益化の鉄則は「**子どもの画面はクリーンに、お金の導線は保護者エリアに集約**」。買うのは子どもではなく親なので、これが効果的かつ安全（COPPA等のルール上も）です。

### 1. 設定（`config.js` を編集するだけ）

```js
window.YOMI_CONFIG = {
  appUrl: 'https://yourname.github.io/yomimochi/', // ← 公開後のURL
  amazon: { enabled: true, tag: 'yourtag-22', domain: 'www.amazon.co.jp' } // ← アソシエイトID
};
```

- **Amazon アソシエイト**: [アソシエイト・プログラム](https://affiliate.amazon.co.jp/)に登録してトラッキングID（`◯◯◯-22`）を取得 → `tag` に入れる。
  - 記録した本の詳細ページに「🛒 この本を かう（おうちの人むけ）」が出ます。ISBN を ISBN-10（=書籍のASIN）に変換し、`amazon.co.jp/dp/<ASIN>?tag=...` で**商品ページ直リンク**。979始まりなど変換不可なものは検索リンクにフォールバック。
  - **必須**: 規約上アフィリエイトであることの表示が必要。「おうちの人へ」ページに開示文を表示済みです。

### 2. 公開（ホスティング）

`localhost` のままでは他人が使えません。静的ファイルだけなので、無料ホスティングにそのまま置けます。

| サービス | 手順 |
|---|---|
| **GitHub Pages** | リポジトリを作って push → Settings > Pages で公開 |
| **Netlify / Cloudflare Pages / Vercel** | このフォルダをドラッグ&ドロップ（またはGit連携）するだけ |

公開URLが決まったら `config.js` の `appUrl` に設定（SNSシェアのリンクに使われます）。

### 3. SNS拡散の仕組み（実装済み）

- **リンクプレビュー（OGP/Twitterカード）**: `index.html` の `<meta>` にタイトル・説明・画像を設定済み。
  - 画像は `ogp.svg` がデザイン元。SNS（特にX）は PNG 推奨なので **1200×630 の `ogp.png` に書き出して**同じフォルダに置いてください（ブラウザで `ogp.svg` を開いてスクショ、または [svgexport](https://www.npmjs.com/package/svgexport) / オンライン変換でOK）。
- **シェアボタン**: 「おうちの人へ」ページに LINE / X / リンクコピー、端末対応時は OS のシェアシート（`navigator.share`）。
- **がんばり自慢シェア**: スタンプ獲得お祝いに「📲 がんばりを シェア」。`○○は いままで ◯さつ よみました！ #よみもち` の文＋URLを共有 → **保護者が誇らしく拡散 → 自然な集客**につながります。

### 収益チャネルを増やすには

今は Amazon のみ。`config.js` と「おうちの人へ」ページに、楽天ブックスアフィリ・応援（投げ銭）・保護者エリア限定の広告などを後から追加できます（子ども画面には出さない方針を維持）。
